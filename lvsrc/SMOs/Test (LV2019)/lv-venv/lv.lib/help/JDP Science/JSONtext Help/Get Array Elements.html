JSONtext.lvlib:Get Array Elements.vi

If a JSON Array, returns all elements of the Array.

If the input JSON is not an Array, this function will return an error, unless "Pass non-arrays" is true, in which case the output will be a one-element array containing the input JSON.

 JSON

Input JSON. Optional "Offset" indicates an offset in this string pointing to the start of the JSON item to operate on.

Offset

Optional Offset in the input "JSON" string. Default is zero, the start of the string. This Offset should point to the start of the JSON Value of interest (or to whitespace in front of it).

error in (no error)

Describes error conditions that occur before this node runs. This input provides standard error in functionality.

    		status

    		code

    		source

Pass non-arrays

If the input JSON is not an Array, this function will return an error, unless "Pass non-arrays" is true, in which case the output will be a one-element array containing the input JSON.

Offset past Array

Offset, in the input JSON, just past the item of interest.

<JSON>Elements

Output JSON. The <JSON> tag indicates to some downstream functions that this is JSON, rather than a standard LabVIEW string.

    		<JSON>

error out

Contains error information. This output provides standard error out functionality.

    		status

    		code

    		source
