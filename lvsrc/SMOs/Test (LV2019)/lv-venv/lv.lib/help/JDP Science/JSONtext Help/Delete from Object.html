JSONtext.lvlib:Delete from Object.vi

This function is DEPRECATED: please use "Delete.vi" instead.

Deletes JSON item at the JSON Path specified. Only accepts paths that specify a unique Object item (no Arrays or multi-item lists), and thus only deletes one item. Use "Delete.vi" to delete multiple items.

 JSON

Input JSON. Optional "Offset" indicates an offset in this string pointing to the start of the JSON item to operate on.

Object Offset

Optional Offset in the input "JSON" string. Default is zero, the start of the string. This Offset should point to the start of the JSON Value of interest (or to whitespace in front of it).

$Path or Item Name

JSON Path notation starts with a $. If not starting with a $, this input is treated as an item name (so <name> and $.<name> are equivalent).

error in (no error)

Describes error conditions that occur before this node runs. This input provides standard error in functionality.
