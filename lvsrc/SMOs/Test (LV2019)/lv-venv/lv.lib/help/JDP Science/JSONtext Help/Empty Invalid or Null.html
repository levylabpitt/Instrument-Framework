JSONtext.lvlib:Empty Invalid or Null.vi

Returns TRUE if the input text contains either

1) no valid JSON

2) any of these JSON types that are "empty":

null

NaN

"" --> empty JSON string

A JSON Object with no items

A JSON Array

Returns False if the input starts with valid non-empty JSON.

 JSON

Input JSON. Optional "Offset" indicates an offset in this string pointing to the start of the JSON item to operate on.

Null, Empty or Invalid

See also JSONtext.
