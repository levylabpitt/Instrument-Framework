JSONtext.lvlib:Colour Items of a JSON string control.vi

Writes the input JSON to a string control and colour the elements for better readability. Note that this may be slow for large JSON structures; consider the following options"

1) Defer Panel Updates set to true will speed up the process by preventing display redraws untill all items are coloured.

2) You may set a limit to number of elements that are actually coloured in order to ensure the VI completes in a reasonable amount of time (the default of -1 means no limit).

 JSON

Input JSON. Optional "Offset" indicates an offset in this string pointing to the start of the JSON item to operate on.

String Control

Control that will be set and coloured.

Maximum UI Property calls

Limits the number of property-node calls.
