JSONtext.lvlib:Read Section Cluster.vim

Set the value of a Section Cluster from a Section of a JSON object. If the Section is not specified, then the data name of the Cluster will be used.

A "Section Cluster" is defined in analogy to the OpenG Variant Config Library "Section Cluster" functions, using a JSON Object, with subObjects representing "Sections", instead of INI format.
