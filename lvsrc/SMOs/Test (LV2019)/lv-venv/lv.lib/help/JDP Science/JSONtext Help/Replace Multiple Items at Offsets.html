JSONtext.lvlib:Replace Multiple Items at Offsets.vi

Replace the JSON Values located at Offsets in the input JSON. The Offsets can be that returned by a previous "Find" operation on the input JSON.

 JSON

Input JSON. Optional "Offset" indicates an offset in this string pointing to the start of the JSON item to operate on.

Offsets of Items to Replace

    		Offset of Item to Replace

New JSON Items

    		<JSON>New Item

error in (no error)

Describes error conditions that occur before this node runs. This input provides standard error in functionality.

    		status

    		code

    		source

<JSON>

Output JSON. The <JSON> tag indicates to some downstream functions that this is JSON, rather than a standard LabVIEW string.

error out

Contains error information. This output provides standard error out functionality.

    		status

    		code

    		source
