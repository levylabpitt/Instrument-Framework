JSONtext.lvlib:Unflatten JSONpath Array to Object.vi

Takes arrays of $Paths items and generates a Object containing those $Paths. Does not support multi-point paths.

For example: the $Paths (Items):

$.A.B1 (1)

$.A.B2 ("x")

$.A2 (true)

$.A3.C (123)

produce the JSON: {"A":{"B1":1"B2:"x"},"A2":true,"A3":{"C":123}}

 $Paths

Array of JSON Paths corresponding to the JSON Items.
