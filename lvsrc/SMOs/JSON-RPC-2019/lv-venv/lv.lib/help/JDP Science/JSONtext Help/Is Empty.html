JSONtext.lvlib:Is Empty.vi

Returns TRUE if the input text starts with any of these JSON types that are "empty":

1) null

2) NaN

3) "" (an empty JSON string)

4) A JSON Object containing no items

5) An empty JSON Array

6) no JSON at all (empty string or only whitespace)

Returns False if the input starts with non-empty JSON.

This function will return FALSE if it detects invalid JSON, but as it only inspects the lead characters, it cannot be relied on to validate the input as proper JSON.

JSON

Input JSON. Optional "Offset" indicates an offset in this string pointing to the start of the JSON item to operate on.

Empty

See also JSONtext.
